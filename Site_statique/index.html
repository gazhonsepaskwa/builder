Monopoly Builder

Le Jeu

Monopoly Builder est l'édition Monopoly dans laquelle les joueurs construiront les bâtiments sur Monopoly Island. Les joueurs utiliseront différents types de ressources pour construire de nombreux étages (étages) de bâtiments. Les joueurs rassembleront et échangeront des ressources de manière stratégique pour construire rapidement. Les joueurs construisent pour accumuler des points et marquer. Ce jeu peut être joué avec 2 à 4 joueurs.
